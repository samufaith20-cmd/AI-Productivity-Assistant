# AI-Powered Productivity Assistant

## 📌 Overview
This project is an AI-based assistant that helps automate daily productivity tasks such as:
- Email generation
- Meeting summarization
- Task planning
- Research assistance
- Chatbot interaction

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API (optional)

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📂 Features
- Easy-to-use interface
- Multiple productivity tools in one place
- Extendable modules
